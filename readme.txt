说明：
中文信息分类识别，基于自定义规则进行分类识别
本应用主要做规则解析。

//
你可以直接运行main.go，开启WEB 80服务，进行规则测试，用于开发调试。
//
在生产环境使用目录规则加载，
调用LoadRulesByDir加载一个目录下的所有规则，支持多级规则目录（多级目录时，开发应当对返回结果处理）
//


规则介绍：<br>
 示例:招标+(公示|公告|通知)<br/>
 解释:识别信息中有招标并且有（公示或公告或通知）的，认为满足此分类<br/>
 语法:支持以下符号+与的关系，|或的关系，^不包含的关系，可以用（）小括号，组织更复杂的表达式。<br/>
 如:招标+(公告|公示|通知)^(中标+(结果|公示))^(作废|流标|无效)<br/>
